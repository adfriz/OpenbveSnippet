# 📄 OpenBVE Route CSV Prefix List

List of available prefixes for the Route CSV format (`.csv`).

| Prefix | Command | Description |
| :--- | :--- | :--- |
| `opt.len` | `Options.UnitOfLength` | Sets the unit of length (meters). |
| `opt.spd` | `Options.UnitOfSpeed` | Sets the unit of speed (km/h). |
| `opt.vis` | `Options.ObjectVisibility` | Sets object visibility distance. |
| `rou.com` | `Route.Comment` | Sets route description. |
| `rou.img` | `Route.Image` | Sets preview image. |
| `rou.tt` | `Route.Timetable` | Loads timetable image. |
| `rou.amb` | `Route.AmbientLight` | Sets ambient light (RGB). |
| `rou.dir` | `Route.DirectionalLight` | Sets directional light (RGB). |
| `rou.ldir` | `Route.LightDirection` | Sets light direction angles. |
| `trn.fld` | `Train.Folder` | Sets default train folder. |
| `trn.vel` | `Train.Velocity` | Sets initial train velocity. |
| `str.rail` | `Structure.Rail.Load` | Loads a rail object. |
| `str.grnd` | `Structure.Ground.Load` | Loads a ground object. |
| `str.free` | `Structure.FreeObj.Load` | Loads a free object. |
| `str.pole` | `Structure.Pole.Load` | Loads a pole object. |
| `trk.rail` | `Track.RailStart` | Starts a new rail. |
| `trk.raile` | `Track.RailEnd` | Ends specified rail. |
| `trk.cur` | `Track.Curve` | Sets curve and cant. |
| `trk.pit` | `Track.Pitch` | Sets vertical gradient. |
| `trk.free` | `Track.FreeObj` | Places a free object. |
| `trk.sta` | `Track.Station` | Defines a station stop. |
| `trk.stop` | `Track.Stop` | Defines stop positions. |
| `trk.sig` | `Track.Signal` | Places a signal. |
| `trk.lim` | `Track.Limit` | Sets speed limit. |
| `trk.beac` | `Track.Beacon` | Places a safety beacon. |
| `trk.bri` | `Track.Brightness` | Sets track brightness. |
| `trk.fog` | `Track.Fog` | Sets fog conditions. |
| `trk.ann` | `Track.Announce` | Plays announcement sound. |
| `trk.buf` | `Track.Buffer` | Places a track buffer. |
